# isc-lease-tool
Tool to display ISC-DHCP-SERVER active (valid) leases

**Requirements:**

- ISC-DHCP-SERVER
- Python 2.7


**Functions:**

Return active (valid) DHCP leases from defaul location 
'/var/lib/dhcp/dhcpd.leases'
